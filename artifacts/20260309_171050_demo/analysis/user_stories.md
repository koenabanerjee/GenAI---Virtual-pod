## US-001 - As a teacher, I want to be able to input student attendance data

**Persona:** Teacher  
**Goal:** Input student attendance data for analysis  
**Business Benefit:** Enable the system to analyze attendance data for predicting academic performance  
**Priority:** High

### Acceptance Criteria
- Teacher can input student attendance data for each class session
- Attendance data is stored and can be retrieved for analysis

## US-002 - As a teacher, I want to be able to input student assignment scores

**Persona:** Teacher  
**Goal:** Input student assignment scores for analysis  
**Business Benefit:** Enable the system to analyze assignment scores for predicting academic performance  
**Priority:** High

### Acceptance Criteria
- Teacher can input student assignment scores for each assignment
- Assignment scores are stored and can be retrieved for analysis

## US-003 - As a teacher, I want to be able to view student performance reports

**Persona:** Teacher  
**Goal:** View student performance reports  
**Business Benefit:** Identify students who may need additional support based on performance data  
**Priority:** Medium

### Acceptance Criteria
- Teacher can view student performance reports with attendance, assignment scores, and exam marks
- Reports are generated in a clear and understandable format
- Teacher can filter reports by student, class, or time period
