## Test Design for US-002 - Student Attendance Viewing

**Target Module:** `student_attendance_viewing`

### Acceptance Criteria Under Test
- Students can view their attendance records for the current semester
- Students can view their attendance records for previous semesters
- Students can download or print their attendance records

### Test Scope
- Unit validation for generated module import and behavior.
- Output contract validation for dictionary-based response shape.
- Smoke-level integration readiness for SDLC pipeline.
