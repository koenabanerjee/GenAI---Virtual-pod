## Test Design for US-002 - Issue and return books

**Target Module:** `issue_and_return_books`

### Acceptance Criteria Under Test
- As a library staff member, I can check out a book to a member
- As a library staff member, I can check in a book when it is returned
- As a library member, I receive a receipt or confirmation when a book is checked out or returned

### Test Scope
- Unit validation for generated module import and behavior.
- Output contract validation for dictionary-based response shape.
- Smoke-level integration readiness for SDLC pipeline.
