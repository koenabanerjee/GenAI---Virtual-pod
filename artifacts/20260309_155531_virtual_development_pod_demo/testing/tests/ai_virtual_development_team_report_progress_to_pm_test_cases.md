# Test Case Document - US-005: AI Virtual Development Team - Report Progress to PM

## 1. Objective
The objective of this test case is to verify that the AI Virtual Development Team is able to generate and send progress reports to the Project Manager on a regular basis, and that the reports include information on completed user stories, remaining user stories, and unresolved risks.

## 2. Preconditions
- The AI Virtual Development Team has access to the progress reporting system.
- The Project Manager has a valid email address registered in the system.
- User stories and risks have been logged in the system.

## 3. Test Cases

### Test Case 1: Regular Progress Report Generation and Delivery
1. The AI Virtual Development Team logs into the progress reporting system.
2. The team generates a progress report.
3. The report includes completed user stories, remaining user stories, and unresolved risks.
4. The report is sent to the Project Manager's email address.
5. The Project Manager receives the report and verifies the information.

### Test Case 2: Missing Information in Progress Report
1. The AI Virtual Development Team logs into the progress reporting system.
2. The team generates a progress report.
3. The report is missing information on completed user stories, remaining user stories, or unresolved risks.
4. The report is sent to the Project Manager's email address.
5. The Project Manager receives the report and verifies the missing information.
6. The AI Virtual Development Team is notified of the missing information and corrects it in the system.
7. A new progress report is generated and sent to the Project Manager.
8. The Project Manager verifies that the new report includes all required information.

## 4. Exit Criteria
- The AI Virtual Development Team is able to generate and send progress reports to the Project Manager on a regular basis.
- The reports include information on completed user stories, remaining user stories, and unresolved risks.
- The Project Manager receives and verifies the reports.
- The system correctly handles missing information in progress reports and allows the AI Virtual Development Team to correct it.
